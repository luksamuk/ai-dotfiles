* Repositório de Modelos de IA Locais

Este repositório contém scripts e configurações para gerar versões especializadas de modelos de linguagem instalados localmente, bem como configurações para ferramentas que utilizam IA.

** Estrutura do Repositório

- =modelfiles/=: Pasta com arquivos .modelfile para criação de modelos especializados com Ollama
- =configs/=: Configurações para ferramentas de IA como OpenWebUI e OpenCode
- =scripts/=: Scripts auxiliares para gerenciamento de modelos

** Como Usar

Para criar todos os modelos configurados:
#+BEGIN_SRC bash
make all
#+END_SRC

Para criar um modelo específico:
#+BEGIN_SRC bash
make qwen3
#+END_SRC

** Modelfiles

A pasta =modelfiles/= contém arquivos de configuração que facilitam a instalação de modelos importantes no Ollama. Estes arquivos definem especificações e parâmetros para os modelos, otimizando o processo de instalação local.
