* Repositório de Modelos de IA Locais

Este repositório contém scripts e configurações para gerar versões especializadas de modelos de linguagem instalados localmente, bem como configurações para ferramentas que utilizam IA.

** Estrutura do Repositório

- =modelfiles/=: Pasta com arquivos .modelfile para criação de modelos especializados com Ollama
- =configs/=: Configurações para ferramentas de IA como OpenWebUI, OpenCode e ~ask-ai-rs~
- =llama-swap/=: Configuração do llama-swap com múltiplos modelos GGUF para RTX 3050 6GB
- =scripts/=: Scripts auxiliares para gerenciamento de modelos

** Como Usar

*** Ollama Modelfiles

Para criar todos os modelos configurados:
#+BEGIN_SRC bash
make all
#+END_SRC

Para criar um modelo específico:
#+BEGIN_SRC bash
make qwen3
#+END_SRC

*** llama-swap (Modelos GGUF)

O llama-swap permite rodar múltiplos modelos GGUF com swap automático baseado em demanda.

**** Download de Modelos

#+BEGIN_SRC bash
cd llama-swap
./download-models.sh qwopus-4b   # Modelo específico
./download-models.sh all          # Baixar todos
#+END_SRC

**** Iniciar Serviço

#+BEGIN_SRC bash
systemctl --user restart llama-swap
curl http://127.0.0.1:12434/v1/models
#+END_SRC

Ver documentação completa em =llama-swap/README.md=.

** Modelfiles

A pasta =modelfiles/= contém arquivos de configuração que facilitam a instalação de modelos importantes no Ollama. Estes arquivos definem especificações e parâmetros para os modelos, otimizando o processo de instalação local.

** Orientações

Este repositório é orientado para a execução de modelos instalados localmente com Ollama e llama-swap, proporcionando configurações e scripts para gerenciar esses modelos de forma eficiente.

*** Nota importante

O script ~ask-ai.py~ está depreciado e não será mais mantido. Estou deixando-o aqui apenas a título de histórico. O projeto ~ask-ai-rs~ oferece uma alternativa robusta em formato de agente com ferramentas RAG.
